## Simple Zombie Game
A quick zombie shooter I've created back in late 2025, when I wanted to learn how to make JS projects. Note
that a lot of the code is all jumbled up and unorganized, since I'm not good at programming.
## What does it have?
- Zombies, of course
- Also a counter showing how many zombies you killed
- 3 different weapons (that you can only change in the main.js file)
## Controls
- WASD to move (oh and your aim changes based on which direction you move in)
- E to fire
